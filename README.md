## 抖音视频下载
### 使用方式
```
# 安装依赖
pip install -r requirements.txt
# 启动脚本
python main.py
```
根据提示进行登录，登录状态将保存到`./douyin.storage`中，登录有效期内再次启动不需要重新登录。
根据提示将视频的分享链接粘贴到运行窗口中回车即可。
### 实现原理
通过[playwright](https://playwright.dev/)模拟浏览器打开视频的分享链接，截获相关的响应包，解析出视频链接进行下载到本地`./videos`目录下。
